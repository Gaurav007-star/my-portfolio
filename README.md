## 💻 Running this Portfolio Locally

To run this portfolio application on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Gaurav007-star/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
